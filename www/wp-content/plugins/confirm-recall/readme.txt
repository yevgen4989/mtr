== Description ==
This is a lightweight WordPress plugin that implements an extra confirmation dialogue between your click on the *Recall* button. Never Recall by accident again!

== Installation ==

1. Download and extract plugin files to a folder in your wp-content/plugin directory.
2. Activate the plugin through the WordPress admin interface.
3. Change the default message through the 'Recall Confirm Message' menu in WordPress.
